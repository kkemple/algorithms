Trees

Joins: 0 Attempted Joins: 0

This is an example of a weighted union tree structure. As connections are created the smaller forests are added to the larger forests giving a short, fat tree structure. This greatly reduces the amount of levels that need to be traversed when searching for nodes.

A successful join is measured by two unconnected nodes being joined at their roots. An attempted join means that the join was not possible because the nodes are already connected.

The Auto button will run a random number of joins by choosing two random nodes to join. Otherwise you can add numbers to the two input fields and join them yourself. There 60 nodes available.
